# قاعدة معرفة التقوية الإنتاجية - المفتاح الشهري

موقع قاعدة معرفة تفاعلية وقابلة للبحث بناءً على خطة التقوية الإنتاجية لموقع "المفتاح الشهري".

## المميزات

- **11 قسمًا شاملاً** تغطي جميع جوانب التقوية الإنتاجية
- **بحث فوري** عبر جميع الأقسام والعناصر
- **تتبع التقدم** مع مؤشرات حالة لكل عنصر
- **عناصر قابلة للتوسيع** مع تفاصيل وطرق التحقق
- **تصميم RTL** كامل باللغة العربية
- **تصميم "الخزنة التقنية"** بألوان فحمية وذهبية

## التقنيات المستخدمة

- React 19
- Tailwind CSS 4
- Framer Motion
- TypeScript
- Vite

## التشغيل

```bash
cd hardening-kb
pnpm install
pnpm dev
```
